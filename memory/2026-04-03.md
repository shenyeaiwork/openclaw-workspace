# 2026-04-03 学习笔记

## Claude Code 架构深度学习

### 核心架构
- **运行时**: Bun (JS/TS运行时+打包器)
- **UI**: React + Ink (终端UI框架)
- **语言**: TypeScript 5.x
- **AI SDK**: @anthropic-ai/sdk
- **MCP SDK**: @modelcontextprotocol/sdk

### 核心模块 (src/)

| 模块 | 说明 |
|------|------|
| `entrypoints/` | CLI/MCP/SDK入口点 |
| `tools/` | 30+内置工具实现 |
| `commands/` | 70+斜杠命令 |
| `services/` | API/MCP/OAuth/Analytics |
| `utils/` | 权限、swarm、shell、model等 |
| `plugins/` | 插件系统 |
| `skills/` | Skills体系 |
| `bridge/` | 远程桥接(WebSocket/SSE) |
| `memdir/` | 持久化记忆管理 |

### 关键设计模式

1. **工具抽象 (Tool)**
   - 每个工具有: name, description, inputSchema, handler
   - 权限上下文: ToolPermissionContext
   - 进度回调: ToolProgressData

2. **Feature Flag系统**
   - 使用`feature('NAME')`控制功能开关
   - 支持条件编译和按需加载

3. **权限系统**
   - 细粒度权限控制
   - 权限模式: default/bypassPermissions/alwaysAsk
   - 规则引擎: alwaysAllow/alwaysDeny/alwaysAsk

4. **多智能体Swarm**
   - 支持Tmux/iTerm2后端编排
   - Agent间消息传递
   - 权限桥接(leaderPermissionBridge)

5. **任务系统**
   - TaskCreate/List/Get/Update/Stop
   - 支持子任务和任务输出

### 学习重点 - 可借鉴的设计

1. **工具注册模式**: 集中注册+条件导入(dead code elimination)
2. **权限上下文传递**: 贯穿所有工具调用
3. **Feature Flag**: 优雅的按需加载
4. **记忆系统**: MEMORY.md truncation机制
5. **命令系统**: 斜杠命令+子命令架构
